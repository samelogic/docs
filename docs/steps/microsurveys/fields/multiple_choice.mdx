Multiple-choice fields are used for collecting answers where the user has to choose from a list of options.

There are different types of multiple-choice templates to select from:

- Yes/No - simple two option choices.
- Thumbs Up/Down - same as Yes/No but uses icons.
- Emojis - smileys icons used to gather emotional sentiment.
